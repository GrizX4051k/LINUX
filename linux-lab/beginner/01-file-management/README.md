# 01 – File Management

Commands: `ls`, `cp`, `rm`, `mv`, `cat`, `mkdir`, `rmdir`, `nano`, `vi`.

Goal: move around the filesystem confidently, create/rename/delete files and folders, and edit basic text files from the terminal.

# Notes – File Management

- `pwd` – print current directory.
- `ls`, `ls -l`, `ls -a` – list files (basic, detailed, including hidden).
- `cd dir`, `cd ..`, `cd` – change directory.
- `mkdir dir`, `rmdir dir` – create and remove empty directories.
- `cp src dest`, `cp -r src/ dest/` – copy files and folders.
- `mv src dest` – move or rename.
- `rm file`, `rm -r dir` – remove (careful with `-r`).
- `cat file` – view file contents.
- `nano file`, `vi file` – edit text files from terminal.
